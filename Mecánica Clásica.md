# Coordenadas generalizadas
Las coordenadas generalizadas son una serie de parámetros que se utilizan para describir el estado de un sistema dependiendo de este.
Por ejemplo, en algunos sistemas se utilizarán **coordenadas cartesianas** $(x,y,z\dots)$ mientras que en otro puede que se utilicen **coordenadas polares** $(r,\theta)$.
Las **velocidades generalizadas** son las derivadas con respecto del tiempo de las coordenadas generalizadas.

Las coordenadas generalizadas se suelen elegir con el objetivo de minimizar la cantidad de coordenadas independientes.
# Espacio de configuración
En mecánica clásica, el espacio de configuración de un sistema es aquel que está determinado por el sistema de coordenadas utilizado.

Por ejemplo, una partícula moviéndose en un espacio Euclidiano en 3 dimensiones cuyas coordenadas son $q=(x,y,z)$, el espacio de configuración del sistema sería $Q=\mathbb{R}^3$, ya que los vectores están representados por 3 números reales.
# Grados de libertad
En mecánica clásica, los grados de libertad son el número de parámetros independientes que definen el estado del sistema.
